# Hangman Game

## CodeAlpha Python Programming Internship - Task 1

### 📖 Description
This project is a console-based Hangman Game developed in Python as part of the CodeAlpha Python Programming Internship. The game randomly selects a word from a predefined list, and the player must guess the word one letter at a time. The player is allowed a maximum of six incorrect guesses before the game ends. The program also includes a visual Hangman drawing, input validation, and an option to play multiple rounds.

---

## 🚀 Features

- Random word selection
- Text-based Hangman game
- ASCII Hangman drawing
- Five predefined words
- Maximum of 6 incorrect guesses
- Displays guessed letters
- Input validation
- Win and lose conditions
- Play Again option

## 🎮 Gameplay

- The game randomly selects a word.
- Guess one letter at a time.
- Correct guesses reveal the letters.
- Wrong guesses draw the Hangman figure.
- You lose after 6 incorrect guesses.
- You win if you guess the complete word.
- After the game ends, you can choose to play again.
